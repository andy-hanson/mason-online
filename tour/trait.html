trait allows multiple inheritance. A Trait is basically a collection of methods, so the body of a Trait looks just like a class body.

Traits also work great as interfaces.

Declare a class to implement a trait with class trait Foo. This copies the methods from the trait to the class.

Traits can have super-traits.

Just like for classes, static methods of traits are inherited.

A = trait
	static
		get static
			0
	get foo
		1
B = trait A
	get bar
		2
C = class
	get baz
		3
Bar = class extends C trait B
bar = new Bar
. Bar.static
. bar.foo
. bar.bar
. bar.baz

trait!
